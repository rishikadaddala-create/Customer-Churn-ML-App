# Customer Churn Prediction System

## 🚀 Live Demo
🔗 [Try the app online](https://customer-churn-ml-app-eg5pothtm8vstixmtjptzv.streamlit.app/)

## 📖 Project Description
This project predicts whether a customer is likely to churn (stop using a service) based on behavioral, demographic, and service-related factors. It uses Python, Pandas, and Scikit-Learn for data analysis and machine learning, with a Streamlit app for interactive predictions.

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-Learn
- Streamlit
- GitHub

## 📁 Files in the Repository
- `app.py` → Streamlit app  
- `churn_model.pkl` → Trained machine learning model  
- `requirements.txt` → Python dependencies  
- Dataset & notebook (optional)

## 💡 Usage
1. Clone the repository.  
2. Install dependencies: `pip install -r requirements.txt`  
3. Run locally: `streamlit run app.py`  

Or simply use the [Live Demo](https://customer-churn-ml-app-eg5pothtm8vstixmtjptzv.streamlit.app/)

## 📈 Features
- Input customer details interactively
- Predict churn probability
- User-friendly Streamlit interface
